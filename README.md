# http-cerver
Http Server Writen in C++

##About

Simple Webserver using GNU Sockets.

##Build Instructions

```
~ git clone https://github.com/abdullahwaqar/http-cerver.git
~ cd http-cerver
~ make
```

##Run

`~ ./main 2005`
